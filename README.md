# PrivacyChase Social — v1.0

In-person only. Encrypted always. No servers ever.

## Architecture

```
Phone = source of truth
PC    = mirror/display device
```

**Phone creates account → PC detects via QR → PC mirrors phone**

## Project Structure

```
privacychase-social/
├── web/
│   └── index.html          ← Deploy to social.privacychase.com
├── capacitor/
│   ├── package.json
│   ├── capacitor.config.json

    └── workflows/
        ├── build-ios.yml   ← Builds IPA on push to main
```

## Web Deployment

1. Push `web/index.html` to your `privacychase-social` repo
2. Cloudflare Pages auto-deploys to `social.privacychase.com`
3. Done — no build step needed

## Mobile Setup (First Time)

```bash
cd capacitor
npm install
npx cap add ios
npx cap add android
npx cap sync
```

### iOS
```bash
npx cap open ios
# Opens Xcode — build and run to your device
```

### Android
```bash
npx cap open android
# Opens Android Studio — build and run to your device
```

## GitHub Actions — Automated Builds

Every push to `main` automatically:
- Builds an iOS IPA (unsigned, sideloadable via AltStore/Sideloadly)
- Builds an Android APK (debug, installable directly)

### Optional: Signed builds

Add these GitHub Secrets for signed production builds:

**iOS signing:**
- `IOS_CERTIFICATE_BASE64` — your .p12 cert, base64 encoded
- `IOS_CERTIFICATE_PASSWORD` — .p12 password
- `IOS_PROVISION_PROFILE_BASE64` — your .mobileprovision, base64 encoded
- `KEYCHAIN_PASSWORD` — any random string

**Android signing:**
- `ANDROID_KEYSTORE_BASE64` — your .keystore file, base64 encoded
- `ANDROID_KEY_ALIAS` — key alias
- `ANDROID_KEYSTORE_PASSWORD` — keystore password
- `ANDROID_KEY_PASSWORD` — key password

### Encode files for GitHub Secrets

```bash
# macOS/Linux
base64 -i YourCert.p12 | pbcopy        # iOS cert
base64 -i YourProfile.mobileprovision | pbcopy  # iOS profile
base64 -i privacychase.keystore | pbcopy  # Android keystore
```

### Installing unsigned IPA (no Apple Developer account needed)

1. Download the IPA from GitHub Actions artifacts
2. Install [AltStore](https://altstore.io) on your Mac and iPhone
3. Drag the IPA into AltStore to install
4. Or use [Sideloadly](https://sideloadly.io) — sign in with any Apple ID

### Installing APK on Android

1. Download `app-debug.apk` from GitHub Actions artifacts
2. On your phone: Settings → Security → Unknown Sources → Enable
3. Open the APK file on your phone and install

## How It Works

### Signup (Phone)
1. Open `social.privacychase.com` on your phone
2. Create account with passphrase + optional TOTP
3. Your passphrase derives your cryptographic keypair
4. Same passphrase on any device = same identity

### PC Mirroring
1. Open `social.privacychase.com` on your PC
2. PC shows a QR code (no account = waiting screen)
3. On your phone: Settings → Show Login QR
4. PC scans phone QR (or phone scans PC QR)
5. PC mirrors your phone account instantly

### Connections (In-Person Only)
1. Tap 📷 → point camera at someone's QR
2. Enter their name and where you met
3. Connected — no bots possible

### P2P Sync
- Uses WebTorrent with public trackers
- Both devices derive same magnet link from your public key
- Data syncs automatically when both devices are online
- Encrypted blobs only — trackers see nothing

## Encryption

- **Key derivation:** PBKDF2 (200,000 iterations, SHA-256)
- **Encryption:** AES-256-GCM
- **Storage:** IndexedDB (encrypted at rest)
- **Transport:** Encrypted blobs only
- **2FA:** TOTP (RFC 6238) — Google Authenticator compatible
